# 📋 Common Tasks & Patterns

## 🛠️ Adding a New Supabase Table
1. Definir el esquema en un archivo `.sql`.
2. Incluir políticas RLS (Row Level Security).
3. Documentar el cambio en `memory/architecture.md`.

## 🎨 Implementing a New UI Component
1. Seguir el patrón de diseño "Bento Grid".
2. Usar variables de CSS definidas en `assets/css/globals.css`.
3. Asegurar que sea mobile-first (icons no muy grandes, touch targets accesibles).
