### 🛠️ Cómo se utilizan:
- **Obsidian**: Permite que el usuario vea el "Cerebro" del proyecto en tiempo real.
- **GSD**: Asegura que antes de escribir código, haya un plan sólido para no perder la estética.
- **Memory Bank**: Evita que olvide decisiones previas sobre el diseño o el backend.
- **Context Engineering**: Optimiza el uso de tokens y mantiene la información organizada.
