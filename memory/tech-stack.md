# 🛠️ Tech Stack: EcoGuía SOS

## Languages & Frameworks
- **HTML5 / CSS3**: Vanilla para máximo control estético.
- **JavaScript (ES6+)**: Lógica de cliente y manipulación del DOM.
- **Vite**: Build tool y servidor de desarrollo.

## Infrastructure & Tools
- **Supabase**: Base de datos PostgreSQL y autenticación.
- **Vercel**: Hosting y despliegue continuo.
- **Cloudflare**: DNS y proxy de seguridad.
- **Obsidian**: Gestión de conocimiento externa (Brain Sync).

## Development Setup
- `npm run dev`: Inicia el servidor de desarrollo local.
- `npm run build`: Genera el bundle de producción.
- **Memory Bank**: Automatizado mediante Antigravity.
